# Name Finder

A script that automatically generates domain name ideas out of combinations of English words.

Only domain names that are available with the .io extension, as well as on Facebook and Twitter, are produced.

## Example:

```
$ yarn start
masterasleep.io
availablemain.io
scientiststore.io
plantproblem.io
educationneck.io
engineguide.io
lessonnatural.io
swaminclude.io
sleepworried.io
principalgetting.io
recordcabin.io
fruitserve.io
raysphysical.io
myselfsoldier.io
...
```

## How to use:

```
git clone https://github.com/fwouts/namefinder.git ~/namefinder
cd ~/namefinder
yarn install
yarn start
```
